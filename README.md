# Kilo HQ // Focus Station

Personal focus site I keep open at work. Matrix rain, binaural beats, no MP3s, no dependencies.

## What it does

- Matrix rain with my name baked into the character set
- 4 generated focus tracks (all built with Web Audio API — zero audio files)
- Click anywhere to send a ripple through the rain
- Color themes that update the whole UI accent
- Volume control, live clock, pomodoro timer
- Waveform visualizer that reacts to whatever track is playing
- Keyboard shortcuts so you never have to reach for the mouse

## Tracks

| Track | Frequency | Good for |
|---|---|---|
| 40Hz Gamma | 40Hz binaural | Memory, cognitive sharpness |
| Alpha Wave | 10Hz binaural | Relaxed alertness, absorbing info |
| Beta Focus | 20Hz binaural | Active concentration, problem solving |
| Deep Focus | 14Hz binaural | Sustained grounded focus |

> Use headphones. Binaural beats only work with separate left/right channels.

## Controls

| Input | Action |
|---|---|
| Click track button | Play / stop |
| Click again | Stop current track |
| Space | Play/stop |
| 1 / 2 / 3 / 4 | Switch tracks |
| C | Cycle through colors |
| Click canvas | Character burst explosion |
| Pomodoro (top right) | Click to start/pause · Double-click to reset |

## Stack

Just an `index.html`. No dependencies, no build step, no npm, nothing.
Hosted on GitHub Pages.
