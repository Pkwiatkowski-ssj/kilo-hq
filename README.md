# Kilo HQ // Focus Station

Personal focus/music site I keep open at work. Matrix rain, binaural beats, no distractions.

## What it does

- Matrix rain with my name baked into the character set
- 4 generated focus tracks (no MP3s — all built with Web Audio API)
- Color themes for the rain
- Volume control

## Tracks

| Track | Frequency | Good for |
|---|---|---|
| 40Hz Gamma | 40Hz binaural | Focus, concentration |
| Alpha Wave | 10Hz binaural | Relaxed alertness |
| Theta Drift | 6Hz binaural | Deep creative flow |
| Deep Focus | 14Hz binaural | Grounding, steady work |


## Stack

Just an `index.html`. No dependencies, no build step, no npm, nothing.
Hosted on GitHub Pages.

## Usage

Hit a track button to start. Hit it again to stop. Adjust volume with the slider. Pick a color. Vibe.
